# Демо-стенд по Docker и CI

## Описание основного проекта

- **app** - каталог с приложеним
    - __init__.py" - служебный init-файл
    - main.py - основной файл проекта,  интерфейс и бэкенд приложения
    - nlp.py - nlp-функционал для работы сервиса

- **tests** - каталог с unit-тестами
    - test_nlp.py - unit-тесты для тестирования модуля nlp.py

- **requirements.txt** - список необходимых библиотек
